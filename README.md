# 🌤️ 智能天气穿搭助手

输入城市名称，自动获取天气数据、分析天气状况、绘制温度/降水趋势图，并给出穿搭建议。

## 功能

- ✅ 获取当前天气和未来7天预报
- ✅ 分析天气状况（温度、湿度、风速、降水等）
- ✅ 绘制一周温度趋势图和降水图
- ✅ 智能穿搭建议（根据温度、天气、季节）

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行

```bash
python weather_agent.py
```

### 3. 输入城市名

```
请输入城市名称（如：北京、上海、广州）: 北京
```

## 使用说明

**方式一：使用 Open-Meteo（推荐，免费无需注册）**
- 直接运行，程序会自动使用免费天气API

**方式二：使用和风天气（需要API Key）**
1. 注册 https://dev.heweather.com/
2. 获取 API Key
3. 编辑 `weather_agent.py`，将 `HEFENG_API_KEY = "YOUR_API_KEY_HERE"` 改为你的Key
